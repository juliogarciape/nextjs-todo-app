# Todo App :writing_hand:

**A simple todo app built with Next.js 14 and Tailwind CSS**

## Installation :package:

```bash
git clone https://github.com/juliogarciape/nextjs-todo-app.git
cd nextjs-todo-app
npm install
npm start
```
